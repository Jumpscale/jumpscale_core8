# Repo Delete

Delete the AYS repo.

```shell
Usage: ays repo delete [OPTIONS]

  Delete EVERY THING in the Repo (DANGEROUS) all INSTANCES and BLUEPRINTS
  will be lost !!!

  make sure to do a commit before you do a delete, this will give you a
  chance to roll back.

Options:
  --help  Show this message and exit.

```
