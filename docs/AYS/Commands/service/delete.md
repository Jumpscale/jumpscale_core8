# ays service delete

```shell
ays service delete [OPTIONS]

  Delete a service and all its children Be carefull with this action, there
  is no come back once a service is deleted.

Options:
  -r, --role TEXT  optional role for the service you want to delete
  -n, --name TEXT  optional name of the service
  --ask            ask confirmation before delete services
  --help           Show this message and exit.
```
