# ays run show

```shell
ays run show [OPTIONS]

  show detail of a run.

  If the --key option is not given, show the detail of the last run. if
  --logs option is given, also show the logs of each job.

Options:
  -k, --key TEXT  key of the run to show
  -l, --logs      show logs of the jobs
  --help          Show this message and exit.
```
